# Weather Forecast Dashboard

A responsive weather application that provides real-time weather information using the OpenWeather API. Users can search for any city and view current weather conditions, temperature, humidity, and wind speed.

## Features

- Search weather by city name
- Real-time weather data
- Temperature display
- Humidity information
- Wind speed details
- Dynamic weather icons
- Error handling for invalid city names
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API
